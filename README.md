# RRB NTPC Typing Test Practice

RRB NTPC Typing Test Practice tool for Railway Non-Technical Popular Categories typing exam preparation.

## Features

- Bilingual typing (Hindi & English)
- English target: 35 WPM | Hindi target: 30 WPM
- Timer based practice (2,5,7 Minutes)
- Instant result with WPM, accuracy, errors
